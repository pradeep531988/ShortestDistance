This is a simple project developed to demonstrate End To End application development with the help of Spring Boot, Thymeleaf, Derby DB, JPA, CSS, Javascript, Data Table and ORM concepts.

This application tried to solve the Shortest path from SRC to Destination. All the Database CRUD operations are exposed through REST webservices.
